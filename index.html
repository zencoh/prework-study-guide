HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title for the webpage, this is extremely important for search engine optimization.
  • The h1, h2, h3, h4, h5, h6 element represent the level of heading to help organize the webpage.
  • The p element is used for paragraphs or blocks of text.
  • The ul, ol, li elements represent unordered lists, ordered lists, and listed items

Git

  • ls: list files and directories
  • cd: change directories
  • cd ..: navigate backwards one directory at a time
  • pwd: print working directory
  • mkdir: create directory
  • touch: create new file
  • git status: check current branch
  • git checkout -b branch-name: creates a new branch and switches to it
